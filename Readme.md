#react-native-xkcd
> Loads a random [XKCD](http://xkcd.com/) comic with caption.

__Usage:__

Plain and simple as of now:

```sh
$ npm install --save react-native-xkcd
```

```js
var XKCD = require('react-native-xkcd');
<XKCD />
```
